中文编程有三大欲望，中文文档、中文软件、中文关键字。而在这三大欲望当中，因为中文文档是无需考虑程序语言文法的，所以，满足中文文档的行为，在这三者中，优先性是第一位的。如果能在编写中文文档的过程中，完全地使用了中文，也能使中文编程语言的拥趸无比愉快。而在现实生活中，存在着对于这种快感执着追求的人，我们通常把这种人称之为美食家。而本仓库，则专门为那些厌倦常见英文单词的人，量体裁衣，提供符合他们目的的美食。
